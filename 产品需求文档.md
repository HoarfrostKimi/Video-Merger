# 视频合并APP - 产品需求文档

---

## 一、项目概述

### 1.1 项目背景

用户使用直播录制软件录制直播视频，录制下来的文件通常是**FLV格式**的分段文件（每段约10-30分钟），需要将这些分段文件合并成一个完整的MP4视频文件。

> **文件格式已确认**：用户确认录制文件为 **FLV格式**（`.flv`）。FLV是直播录制领域最常用的视频格式（录播姬、BililiveRecorder等均采用此格式）。

### 1.2 目标用户

- **技术小白**：不懂编程技术，需要简单易用的工具
- **直播录屏用户**：使用录播姬、BililiveRecorder等软件录制直播
- **Windows用户**：仅需支持Windows操作系统

### 1.3 核心价值

1. **自动扫描**：自动发现文件夹中的分段视频文件
2. **格式转换**：将FLV格式转换为更通用的MP4格式
3. **视频合并**：将多个分段文件合并成一个完整视频
4. **简单易用**：无需专业知识，一键操作

## 二、核心功能需求

### 2.1 功能清单

| 功能模块 | 功能描述 | 优先级 |
|---------|---------|--------|
| 文件夹扫描 | 选择文件夹后自动扫描所有FLV分段文件 | 高 |
| 文件列表展示 | 展示扫描到的所有视频文件，按时间排序 | 高 |
| 格式转换 | 将FLV文件转换为MP4格式 | 高 |
| 视频合并 | 将多个分段文件合并成一个完整MP4视频 | 高 |
| 批量处理 | 支持批量选择文件进行转换/合并 | 高 |
| 进度显示 | 显示转换/合并进度 | 高 |
| 输出设置 | 设置输出目录、文件名等 | 中 |
| 日志记录 | 记录处理过程和结果 | 低 |

### 2.2 详细功能描述

#### 2.2.1 文件夹扫描

- 用户点击"选择文件夹"按钮
- 自动扫描该文件夹及其子文件夹
- 识别所有FLV格式的文件
- 按文件名中的时间戳或序号自动排序
- 显示文件数量、总时长等信息

#### 2.2.2 文件列表展示

- 以列表形式展示所有扫描到的视频文件
- 显示文件名、文件大小、时长、创建时间
- 支持多选文件
- 支持按时间顺序自动分组（同一时间段录制的文件）

#### 2.2.3 格式转换

- 将单个FLV文件转换为MP4格式
- 保留原视频质量
- 支持自定义输出路径

#### 2.2.4 视频合并

- 选择多个分段文件后，按顺序合并
- 合并后输出为单个MP4文件
- 自动检测文件顺序（基于文件名）
- 支持自定义输出文件名

#### 2.2.5 批量处理

- 一键选中所有扫描到的文件
- 批量转换或合并
- 支持取消正在进行的操作

## 六、非功能需求

### 6.1 性能需求

- 扫描100个文件的文件夹时间不超过5秒
- FLV转MP4的速度不低于原视频时长的1.5倍（即2小时视频转换时间不超过1小时20分）
- 合并10个分段文件的时间不超过30秒（不含转码时间）

### 6.2 兼容性需求

- 支持Windows 10及以上版本
- 支持常见FLV编码格式（H.264视频 + AAC音频）
- 支持文件名中包含中文、空格、特殊字符

### 6.3 可用性需求

- 界面操作步骤不超过3步完成核心任务
- 提供清晰的进度指示
- 错误提示简洁明了

- **FLV**：Flash Video，一种流媒体视频格式，广泛用于直播录制

- **分段录制**：直播录制软件为防止文件过大，将视频分成多个文件保存，通常每段10-30分钟
