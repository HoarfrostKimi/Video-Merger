基于 Manifest V3 的轻量级单页扩展，由五个文件组成三个角色：
- `manifest.json` 声明权限（storage、activeTab）及 host_permissions（http://127.0.0.1/*、http://localhost/*），通过 content_scripts 将 `content.js` + `content.css` 注入到 `https://member.bilibili.com/platform/upload/video/frame*` 页面，action 默认弹出 `popup.html`。
- `content.js` 是核心自动化层：自执行 IIFE 防止重复注入，先创建悬浮面板 UI，再暴露 `uploadAndSubmit` / `batchUpload` / `uploadViaAddPart` 等步骤函数，按顺序驱动 B 站 Vue 表单（查找 file input → 等待标题输入框 → 设置创作声明/可见范围 → 截帧设封面 → 点击立即投稿）；同时监听 URL 参数 `autoFiles`，从本地服务器地址拉取已合并好的 MP4 列表实现与「视频合并 app」联动。
- `popup.js` 仅负责打开 B 站投稿页，把用户引导到被注入脚本的目标页面。
依赖方向单向：popup → 目标页面 → content_script，content_script 通过 DataTransfer + HTMLInputElement.files setter 绕过 Vue 自定义属性定义来触发上传事件。
